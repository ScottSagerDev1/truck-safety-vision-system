# Edge Impulse Setup

## Project Details
- **Project Name**: Truck Safety Vision System
- **Project URL**: https://studio.edgeimpulse.com/studio/833833
- **Target Device**: Raspberry Pi 5

## Workflow
1. Collect training data
2. Upload to Edge Impulse
3. Train model
4. Export model to `models/` folder
5. Deploy to Raspberry Pi
